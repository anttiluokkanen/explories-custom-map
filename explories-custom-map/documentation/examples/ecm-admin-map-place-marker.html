Admin place marker example

This example shows how to place a marker on the map and retrieve it's location with ECMAdminMap.getMarkerLocation() in the admin section.

Click on the map to place a marker and click Get marker location button.

Config options can be set via setConfig method. Here are shown the default values:

ECMAdminMap.setConfig({
    initLocation: {lat: 64.144319, lng: 26.872043},
    mapElId: 'ecm-map',
    mapTypeId: 'roadmap',
    mapInitZoom: 10,
    mode: 'marker',
    route: null
});
